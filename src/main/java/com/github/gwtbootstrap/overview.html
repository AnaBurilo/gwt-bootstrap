Welcome to the GWT-Bootstrap Javadoc.

GWT-Bootstrap provides GWT widgets styled and (partly) operated by Twitter's Bootstrap. Bootstrap is a project of Mark Otto and Jacob Thornton.

Installation

Add 
	{@literal }
to your {@code *.gwt.xml} file.

Setting up UiBinder

Add 
	xmlns:b="urn:import:com.github.gwtbootstrap.client.ui"
to the top of every UiBinder file that uses GWT-Bootstrap widgets. You can then use the widgets in the "b" namespace: 
	{@literal Some Button}

As usual, all setter map to UiBinder attributes. So the method {@link com.github.gwtbootstrap.client.ui.Icon#setType(String type) setType(String iconName)} can (and in this case: has to) be used via: 

	{@literal }
As you can see, the values are the ones of Bootstrap without the prefix ("plane" instead of "icon-plane").

Enabling Responsive Design

Create a {@link com.github.gwtbootstrap.client.ui.resources.ResourceAdapter ResourceAdapter} and let {@link com.github.gwtbootstrap.client.ui.resources.ResourceAdapter#hasResponsiveDesign() hasResponsiveDesign()} return {@code true}. 
	public interface MyResourceAdapter implements ResourceAdapter {
 	
		{@literal @Override}
		public boolean hasResponsiveDesign() {
			return true;
		}
 	}
